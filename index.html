LeanCloud Edge

这里存放一些「非主流」的关于 LeanCloud 技术以及一些另类的用法。主要涉及的是我个人的偏好，比如 TypeScript 以及 .NET 相关的 LeanCloud SDK 的用法以及文档。

LeanCloud TypeScript SDK

之前在官网的 JavaScript 数据存储开发指南 上架了一阵子的 TypeScript SDK 的示例代码和相关文档，但是经过「团队讨论」，最后还是下架了。我个人表示遗憾，那没办法我打算自己维护 TypeScript SDK 的文档，另外，我个人比较喜欢强类型语言，我对 TypeScript 的好感完全来源于 CSharp，但是当我用了一段时间之后，发现 TypeScript 还是很适合作为客户端的开发语言。详细请查看 LeanCloud TypeScript SDK

另外，我个人使用 ionic2 框架开发了几款应用，对 Angular2 以及 Cordova 框架十分喜爱。

LeanCloud .NET Core SDK

感叹于 Parse 开源了 SDK 的源代码实在是写的太好了，我自己也没实力单独写一个能与之媲美的版本，因此在 .NET Core 1.0 发布之后，我用 .NET Core 编译了一下，这样就可以在 ASP.NET Core 里面直接使用 LeanCloud 作为后端存储，开发网站以及服务端的 API（ASP.NET Web Api 大法好）。

FAQ

Q：这些代码都有什么版权限制么？

A：除了 .NET 相关的代码是直接引用 Parse-SDK-dotNET 作为个人技术研究用，其他版权最终归属 LeanCloud 官方。

Q： 为什么这些技术分享不放在官网？

A： 我努力过，但是团队内部还是更偏向于服务主流的开发趋势，比如热火朝天的 Swift，比如 JS 的新标准之类的。但是作为一个文档工程师（你才是程序员！你全家都是程序员！）我也无能为力，但是我还是很执着于影响身边的人去学习或者至少接触一下微软技术圈的东西，这是我能做的一点点小事了。
