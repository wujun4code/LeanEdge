安装

获取 LeanCloud JavaScript SDK

1.源码下载

javascript-sdk

2.npm

  npm install leancloud-storage --save

获取 d.ts 文件

1.源码获取

从 typed-leancloud-storage 下载根目录里面的 index.d.ts 文件即可

2.typings 获取

  # 首先安装 d.ts 管理工具 typings 
  npm install typings --global

  # 执行如下命令行安装
  typings install leancloud-storage --save
